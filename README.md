## Running it locally

The prototype is built with [Svelte](https://svelte.dev/docs/svelte/overview) and TypeScript. 

Prerequisites:
- Pulling a local copy of the project to your computer
- Having an install of [Node.js](https://nodejs.org/en/download) (which usually includes the package manager npm)

After doing the prerequisites, you can install dependencies with 
```sh
npm install
``` 

and then start a development server with
```sh
npm run dev
```

Enter the localhost address from the terminal into your browser. You should see the prototype up and running!
